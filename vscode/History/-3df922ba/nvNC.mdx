# 🌐 Domain DNS Transfer: Hostinger to Cloudflare (Next.js Friendly)

This guide will help you move your domain's DNS from Hostinger to Cloudflare, configure optimal SSL/TLS and performance settings, and ensure compatibility with modern applications.

---

## ⚖️ Benefits Comparison: Cloudflare DNS vs Hostinger DNS

| Feature                       | 🌩️ Cloudflare DNS       | 🏠 Hostinger DNS         |
| ----------------------------- | ----------------------- | ------------------------ |
| Free SSL                      | ✅ Universal SSL        | ❌ Manual Setup Required |
| DDoS Protection               | ✅ Built-in             | ❌ None                  |
| CDN / Global Cache            | ✅ Yes                  | ❌ Not available         |
| IP Masking / Firewall         | ✅ Enabled              | ❌ No                    |
| Auto Performance Optimization | ✅ Brotli, Minify       | ❌ Basic Only            |
| DNS Propagation Speed         | 🚀 Fast                 | 🐢 Slower                |
| Analytics and Rate Limiting   | ✅ Available            | ❌ No                    |
| SEO & Production Ready        | ✅ Strongly Recommended | ❌ Not Optimal           |

👉 **Recommendation:** Always use Cloudflare DNS for production-grade websites.

---

## 1️⃣ Create a Free Cloudflare Account

- Visit: https://dash.cloudflare.com
- Sign up using Gmail or any email

---

## 2️⃣ Add Your Site to Cloudflare

- Click "+ Add a Site"
- Enter your domain (e.g. `naturalsefaa.com`)
- Choose the **Free Plan**

---

## 3️⃣ DNS Records Import or Manual Entry

Cloudflare usually imports your DNS records from Hostinger. If not, add manually:

| Type | Name | Value    |
| ---- | ---- | -------- |
| A    | @    | VPS IPv4 |
| A    | www  | VPS IPv4 |
| AAAA | @    | VPS IPv6 |
| AAAA | www  | VPS IPv6 |

🟠 **Ensure** the **Proxy status** (orange cloud) is **ON** to activate Cloudflare protection.

---

## 4️⃣ Update Nameservers in Hostinger

Cloudflare will give you nameservers like:

```text
josh.ns.cloudflare.com
lola.ns.cloudflare.com
```

🔁 Now go to Hostinger:

1. Login to Hostinger dashboard
2. Navigate to **Domains > DNS / Nameservers**
3. Select **Custom Nameservers**
4. Paste Cloudflare's nameservers
5. Save

🕓 Wait for propagation (15 mins to 12 hours)

🔎 Check status: https://dnschecker.org

---

## 5️⃣ 🧹 Remove Hostinger Default Nameservers

If your domain still has Hostinger's default DNS:

```text
ns1.dns-parking.com
ns2.dns-parking.com
```

👉 Remove them from Hostinger domain panel and **keep only Cloudflare nameservers**.

✅ This avoids conflicts and ensures your DNS is fully managed by Cloudflare.

---

## 6️⃣ Configure SSL in Cloudflare

Navigate: **SSL/TLS > Overview**

- SSL Mode: `Full (strict)`
- Always Use HTTPS: ✅ Enabled
- Automatic HTTPS Rewrites: ✅ Enabled
- TLS 1.3: ✅ Enabled
- Minimum TLS Version: 1.2

💡 If using `Full (strict)`, your VPS must have a valid SSL (e.g. Let's Encrypt).

---

## 7️⃣ Performance Optimization

Go to **Speed > Optimization**:

| Feature                     | Status      |
| --------------------------- | ----------- |
| Auto Minify (HTML, CSS, JS) | ✅ On       |
| Brotli Compression          | ✅ On       |
| Rocket Loader               | ⚙️ Optional |
| Early Hints                 | ✅ Enabled  |
| HTTP/2 Prioritization       | ✅ On       |

---

## 8️⃣ Caching (Next.js Compatible)

Cloudflare → Caching > Configuration:

- Browser Cache TTL: `1 hour` or `4 hours`
- Always Online: ✅ Enabled
- ❌ Do NOT enable "Cache Everything" if your app uses **SSR** or **ISR**

🧠 For **Next.js** apps (App Router / SSR / API routes): dynamic rendering must not be fully cached.

---

## 9️⃣ Security Settings

Cloudflare → Security > Settings:

- DDoS Protection: ✅ Enabled
- Bot Fight Mode: ✅ On
- IP Access Rules: 🌍 Optional (e.g. Geo-blocking)

---

## 🔍 Verify DNS Setup

Use terminal:

```bash
dig naturalsefaa.com
```

✅ You should see a Cloudflare IP like `104.21.x.x`, meaning proxy is active.

---

## 🧑‍💻 Notes for Next.js Users

- ✅ Works with SSG, SSR, App Router, API routes
- 🔐 Use "Full (strict)" only if valid SSL exists on server
- ❌ Avoid "Cache Everything" page rules
- 🧠 Cloudflare = CDN + DDoS + TLS termination. Server handles rendering.

---

## ✅ You’re Done!

Your domain is now:

- ⚡ Speed optimized with CDN & Brotli
- 🔐 Secured with HTTPS & DDoS protection
- 🚀 Ready for Next.js production deployment

**Cloudflare + Next.js = Reliable, fast, and secure web delivery.**
